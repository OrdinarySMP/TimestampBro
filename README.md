# Timestamp Bro 

**Timestamp Bro** adds timestamps to chat messages, so you can see when each message was sent

---

## Features

- **Chat Timestamps** - Adds a timestamp in front of every chat message.
- **Custom Time Format** - Choose your preferred format (e.g., [HH:mm:ss], [HH:mm]).
- **Hover Text** - Optional hover tooltip over the timestamp.
- **Color Customization** - Pick your own timestamp color.
- **Enable/Disable Toggle** - Turn timestamps on or off in-game.

---

## Configuration

All options are available in-game via the **Mod Menu** UI (requires [Mod Menu](https://modrinth.com/mod/modmenu)).
